# Blog Post 초안 작성

WebFlux / Non-blocking I/O 관련 블로그 포스트 초안을 생성합니다.

## 입력
$ARGUMENTS (주제 또는 이슈 번호)

## 수행 작업

`.claude/agents/blog.md`의 Blog Agent 역할을 수행합니다.

1. 입력이 이슈 번호인 경우 해당 이슈와 구현 코드를 확인합니다
2. 입력이 주제인 경우 관련 구현 파일을 탐색합니다
3. **실제 프로젝트 코드 기반**으로 블로그 포스트 초안을 작성합니다:
   - 배경: 왜 이 패턴이 필요했는가
   - Spring MVC vs WebFlux 비교
   - 실제 코드 예제
   - 핵심 개념 설명
   - 주의할 점 / 트레이드오프
4. `/docs/blog/[YYYY-MM-DD]-[주제].md`에 저장합니다

## 출력
- 저장된 파일 경로
- 포스트 제목 제안
- 대략적인 목차
